COMIC CHARACTERS

ABOUT PROJECT


Project Objectives :

  1. Describe a character with categories such as Universe origin, alignment, gender, living status and identity status.
  2. Compare which character is the strongest based on their attributes
  3. Find which character has had the most comic book appearances

Data Sources :
  • Kaggle Dataset : Character Statistics
  • Kaggle Dataset : Marvel / DC Characters & Character Appearances

Data Preparation Notes:
  • For the Marvel and DC characters, there were characters who had a Neutral alignment and characters also had numerous gender types. However for this dataset, I am only looking at male and female characters.

CHARACTER STATISTICS

Using an interactive bar chart with different toggles to filter categories which compares the abilities of different comicbook characters on a scale of 0 - 100 based on the different categories:

  • Intelligence
  • Strength
  • Speed
  • Combat
  • Durability
  • Power
Another category called 'Overall' is an average of the 6 categories listed above to see which character is the most skilled as a whole.

CHARACTER APPEARANCES

With a heatmap, this graph will show you the year a character first appeared in and the intensity of the color will reflect the number of appearances this character has had.
For this dataset, I only used characters who had more than or equal to 500 appearances, as using all data samples for this visualization caused a lot of clutter.

CHARACTER TREE

This graph shows a tree map of the characters in the Marvel and DC universes, with the root and parent nodes having different categories that filter the characters. The different categories that make up the tree in order are :

  1. Universe
  2. Alignment (Whether they are good or bad)
  3. Gender
  4. Alive or Dead ?
  5. Identity Status - Public or Secret ?
